
 📊 AdventureWorks Sales Dashboard (Excel)

 📌 Project Overview

This project presents a fully interactive Sales Performance Dashboard built in Microsoft Excel using the AdventureWorks database.

The raw transactional data was transformed and structured into a Star Schema Data Warehouse model to facilitate efficient analysis and improve reporting performance.

The objective of this project is to simulate a real-world BI workflow:
from raw operational data ➝ data transformation ➝ dimensional modeling ➝ analytical dashboard.


 🏗 Data Modeling & Transformation

Before building the dashboard:

* Performed data cleaning and transformation
* Structured data into a Star Schema
* Created:

  * Fact Table: Sales Fact
  * Dimension Tables: Date, Product, Customer, Territory, Ship Method, etc.
* Ensured proper relationships between fact and dimensions
* Optimized the model to support fast aggregation and filtering

This structure enables scalable analysis and aligns with best practices in Data Warehousing.


 📊 Dashboard Features

The dashboard provides executive-level insights including:

🔹 KPI Summary

 🔹 Analytical Views
